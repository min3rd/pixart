# Keyboard Shortcuts

Pixart supports customizable keyboard shortcuts to help you work faster. You can view and customize all shortcuts from the **Help** menu â†’ **Keyboard Shortcuts**.

## Default Shortcuts

### File Operations

| Action | Default Shortcut |
|--------|-----------------|
| New Project | `Ctrl + N` |
| Open from Computer | `Ctrl + O` |
| Save | `Ctrl + S` |
| Save to Computer | `Ctrl + Shift + S` |

### Edit Operations

| Action | Default Shortcut |
|--------|-----------------|
| Undo | `Ctrl + Z` |
| Redo | `Ctrl + Y` |
| Move Selection Pixels | `Shift + V` |

### Tool Selection

| Tool | Default Shortcut |
|------|-----------------|
| Select Layer | `V` |
| Rectangle Select | `M` |
| Ellipse Select | `Shift + E` |
| Lasso Select | `L` |
| Eyedropper | `I` |
| Fill | `G` |
| Eraser | `E` |
| Line | `Shift + L` |
| Circle | `C` |
| Square | `R` |
| Brush | `B` |
| Bone | `Shift + B` |

### Insert Operations

| Action | Default Shortcut |
|--------|-----------------|
| Insert Image | `Ctrl + I` |

## Customizing Shortcuts

1. Click **Help** in the header menu
2. Select **Keyboard Shortcuts**
3. Click **Edit** next to the action you want to customize
4. Press the key combination you want to use
5. Click **Save** to confirm

### Conflict Detection

The system will automatically detect if your new shortcut conflicts with an existing one and display a warning. You cannot save a shortcut that conflicts with another action.

### Resetting Shortcuts

- To reset a single shortcut: Click the **Reset** button next to the action
- To reset all shortcuts: Click **Reset All to Defaults** at the bottom of the dialog

## Notes

- All shortcuts are saved to your browser's local storage
- Shortcuts persist across sessions
- Use `Ctrl` on Windows/Linux or `Cmd` on macOS for modifier keys
- Shortcuts are case-insensitive for letter keys
